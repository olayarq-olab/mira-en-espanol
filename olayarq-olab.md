# Observatorio de Sesgo antisemita

Overview

MIRA en Español is a media analysis platform that documents and analyzes narratives related to Jewish communities in Spanish-language media.

The project is designed as a structured observatory, combining a database of documented cases with analytical context, with the goal of providing transparency, traceability and clarity.

Project Goals
Build a structured and searchable database of documented cases
Provide contextual analysis in a clear and transparent way
Ensure usability for researchers, journalists and institutions
Maintain a clean, neutral and professional user experience

Key considerations:

Data structure scalability
Security (especially due to the nature of the project)
Maintainability and long-term ownership

Current Status
Prototype developed using Lovable
Frontend built with modern web technologies (React / Next.js)
Initial structure for documented cases and media profiles implemented
Basic filtering system (by media, author, category)

Core Features

1. Documented Cases

Each case includes:
Media outlet
Author (optional)
Publication date
Article title
Excerpt
Analysis
Category (trope)
Source URL
Image / cartoon (optional)
Image caption and credits

2. Media Profiles
Overview of each media outlet
Ownership and structure
Contact details
Archive of related cases

3. Filtering System
Filter by media outlet
Filter by author
Filter by category (trope)
Tech Stack
Frontend: React / Next.js
Backend: Supabase (authentication, database, storage)
Hosting: Lovable (prototype phase)
Version control: GitHub
